Herramientas de Subredes

1. Calculadora CIDR

¿Cómo funciona? El enrutamiento entre dominios sin clases (CIDR) permite una gestión flexible de direcciones IP. La calculadora toma la dirección IP base y el prefijo de red (máscara), realizando las siguientes operaciones a nivel de bits:

  • Máscara y Red: Convierte el prefijo en una máscara binaria de 32 bits y aplica un AND lógico con la IP para hallar la Dirección de Red.
  • Broadcast: Invierte los bits de la máscara (operación NOT) y aplica un OR lógico con la IP de red para encontrar el límite superior o Dirección de Broadcast.
  • Hosts: Determina la cantidad de hosts direccionables mediante la fórmula 2(32 - prefijo) - 2, excluyendo las direcciones de red y broadcast.

2. Calculadora VLSM

¿Cómo funciona? La Máscara de Subred de Longitud Variable (VLSM) optimiza el direccionamiento al segmentar una red principal en bloques de tamaños adaptados a necesidades específicas:

  • Ordenamiento Jerárquico: El algoritmo ordena los requerimientos de hosts de mayor a menor para garantizar que las subredes grandes se asignen primero, previniendo el solapamiento de direcciones.
  • Cálculo de Bits: Para cada subred, busca el menor número de bits de host (n) que cumpla con la condición 2n - 2 ≥ hosts requeridos. El nuevo prefijo será 32 - n.
  • Asignación Consecutiva: Tras calcular los rangos de la primera subred, la dirección IP base se desplaza al siguiente bit disponible inmediatamente después del broadcast anterior, repitiendo el proceso secuencialmente.
